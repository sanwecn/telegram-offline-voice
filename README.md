# Telegram Offline Voice Skill for Moltbot

This skill provides an optimized offline TTS experience for Telegram users.

## Installation
1. Ensure `ffmpeg` is installed on your system.
2. Install the `sherpa-onnx-tts` skill and download a Chinese model.
3. Add this skill to your Moltbot.
