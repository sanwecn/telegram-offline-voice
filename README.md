# Telegram Voice Skill for Moltbot

This skill provides an optimized TTS experience for Telegram users, supporting both high-fidelity cloud voices (Edge-TTS) and local models.

## Update Note (2026-01-29)
- Switched default high-fidelity provider to **Edge-TTS**.
- Added support for the premium **Xiaoxiao** neural voice.
- Maintained "Golden Ratio" pacing tuned by @sanwecn.

## Installation
1. Ensure `ffmpeg` is installed.
2. Install via MoltHub or by cloning this repo.
